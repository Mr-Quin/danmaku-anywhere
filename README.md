> 没有弹幕怎么看番？
> Danmaku Anywhere是一个可以在任何视频网站上加载弹幕的浏览器扩展

## 功能

- 在任何有视频的网页上加载弹幕，彻底解决看番无弹幕的问题
- 从多个弹幕源获取弹幕，目前支持：
  - 弹弹Play
  - B站
  - 腾讯
- 手动导入其他来源的弹幕文件，支持常见的`.xml`格式
- 自动匹配视频弹幕（需配置）
- 可自定义的弹幕样式
- 导出本地缓存的弹幕

# @danmaku-anywhere/danmaku-converter

Utilities for converting between danmaku formats (e.g. XML to internal format).

## Usage

Consumed by `danmaku-engine`, `danmaku-provider`, and `web-scraper`. Main entry: `src/index.js` / `src/index.d.ts`.

## Scripts

| Command | Description |
| ------- | ----------- |
| `pnpm build` | Compile with tsgo |
| `pnpm test` | Vitest |
| `pnpm dev` | Watch build |

## Dependencies

- `xml-js`, `zod`

# @danmaku-anywhere/danmaku-engine

Rendering engine for danmaku overlay. Uses `@mr-quin/danmu` for rendering.

## Usage

Consumed by the browser extension (`packages/danmaku-anywhere`). Depends on `@danmaku-anywhere/danmaku-converter` for format conversion.

## Scripts

| Command | Description |
| ------- | ----------- |
| `pnpm build` | Compile with tsgo |
| `pnpm test` | Vitest |
| `pnpm dev` | Watch build |

## Dependencies

- `@danmaku-anywhere/danmaku-converter`, `@mr-quin/danmu`

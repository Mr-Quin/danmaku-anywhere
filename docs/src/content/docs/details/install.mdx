## Chrome

作者使用且仅使用 Chrome 浏览器，安装说明也基于 Chrome 浏览器，其他浏览器可能会有不同。

### 商店
[Chrome 应用商店](https://chromewebstore.google.com/detail/danmaku-anywhere/jnflbkkmffognjjhibkjnomjedogmdpo?authuser=1&hl=zh)


### 手动安装

1. 从[Github](https://github.com/Mr-Quin/danmaku-anywhere/releases/latest)下载最新版本。
2. 找一个文件夹解压。
3. 然后进入[扩展页面](chrome://extensions/)，启用开发者模式，点击 "加载未打包的扩展" 并选择已解压的扩展文件夹。
4. 后续的更新直接解压到这个文件夹中并覆盖并刷新扩展即可。

:::note
请勿删除解压的文件夹。
:::

### Firefox

相比 Chrome 版本，Firefox 版本缺少一些 API 的支持，并且存在一定的兼容性问题，可能在一些奇怪的地方出现问题，不推荐使用。

Firefox没有上架到商店，但是可以从[Github](https://github.com/Mr-Quin/danmaku-anywhere/releases/latest)下载。

手动安装似乎须使用开发者版本的 Firefox。
参考[这里](https://extensionworkshop.com/documentation/develop/temporary-installation-in-firefox/)和[这里](https://extensionworkshop.com/documentation/publish/signing-and-distribution-overview/)

description: 自动模式
---

自动模式可以自动化一部分操作，提高用户体验：
1. 自动根据视频匹配弹幕
2. 换集自动切换弹幕
3. 关联视频和弹幕


## 支持的网站

自动模式需要对网站进行适配，目前支持的网站有：

- [Plex](https://app.plex.tv/)

:::tip
希望支持更多网站，请提交[功能请求](https://github.com/Mr-Quin/danmaku-anywhere/issues/new)
:::

## 启动

在[装填配置](/details/mount-profile)的适配选项中选择对应的适配。

没有适配的网站请选择”无“，否则可能会影响正常使用。

适配规则是一种从网页中采集番剧信息的方案，用于实现弹幕的自动匹配。理想情况下，适配规则可以完全自动化弹幕的匹配过程，从而免去手动搜索的麻烦。

:::note
目前自动适配固定使用弹弹Play源
:::

适配规则分为两种：
- [AI自动适配](./ai)
- [XPath规则](./xpath)

## 关联适配规则
适配规则可以与一个或多个**装填配置**关联，当装填配置启用时，扩展会自动使用关联的适配规则。

1. 在启用了**装填配置**的网页中，打开控件窗口
2. 点击**适配规则**
3. 打开下拉菜单，选择一个适配规则。
4. 如果没有合适的适配规则，可以点击**添加适配规则**添加一个新的。

:::note
v0.19.0 中新增功能

AI自动适配功能仍在测试中，效果可能不稳定，遇到问题请反馈。
:::

:::caution[安全提示]
- 此功能会向服务器发送网页内容，如果是私人网站请注意隐私。
- 服务器保存7天内的日志，用于调试和分析。
:::

开启AI自动适配后，扩展会从网页中截取可能包含番剧信息的文本，然后用AI从中提取番剧信息。

目前测试下来效果还可以，但是受网页结构影响较大，可能需要修正标题和集数。

**不要在播放非番剧视频时开启AI自动适配。**

## 开启AI自动适配

在适配规则中勾选**使用AI**并保存。

## 注意事项

- 匹配结果存在缓存，同一个网页一定时间内重新匹配会获得相同的结果。
- AI服务存在调用次数限制，可能会出现调用失败的情况，请过一分钟再试。
- 不要在非番剧网站开启AI自动适配。

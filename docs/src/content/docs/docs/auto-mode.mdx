title: 自动模式
description: 自动模式
---

自动模式可以自动化部分操作，提升用户体验：

1. 自动匹配视频弹幕
2. 自动切换集数时更换弹幕
3. 关联视频和弹幕

## 支持的网站

自动模式需要对网站进行适配，目前支持的网站有：

- [Plex](https://app.plex.tv/)

:::tip
希望支持更多网站，请提交[功能请求](https://github.com/Mr-Quin/danmaku-anywhere/issues/new)。
:::

## 启动

在[装填配置](/docs/mount-profile)的适配选项中选择对应的适配。

对于没有适配的网站，请选择“无”，否则可能会影响正常使用。

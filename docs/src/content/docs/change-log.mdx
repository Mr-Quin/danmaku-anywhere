## v1.4.0 - 2025-12-20

## 特性

### 弹幕源

重做弹幕源设置，现在可以配置任意数量的源

除了MacCMS外，所有源都可以进行自动匹配。自动匹配时使用排名最靠上的源。

**弹弹Play**
- 可以配置兼容弹弹Play API的弹幕服务
- 可以使用自己的弹弹Play API key

### 装填配置

- 适配设置（自动化方式）转移至装填配置
- 编辑装填配置时会提示进行自动化的设置
- 暂时移除导入配置的功能
- 添加导入导出分享码的功能（仅限自动化方式为xpath时）

### Xpath配置（原适配配置）

- 装填配置的自动化方式为xpath时可以在网页扩展窗口中进行设置
- AI设置移动至装填配置中
- 移除“仅使用标题匹配”选项，现在会自动判断匹配方式
- 优化xpath配置的编写方式
- 添加了一套内置的regex，可自动识别一些常见的剧集命名方式
- regex改为可选项
- 可以通过分享码分享配置

### 弹幕库

- 重做弹幕库UI，改为树状显示（原有的弹幕库界面暂时保留）
- 可以使用多选功能装填/导出/删除多个剧集

### 搜索

- 优化搜索界面，现在可以在搜索界面快速开关弹幕源/调整顺序

### 弹幕过滤

- 过滤功能移动至单独的页面（之前在弹幕设置中）
- 优化页面逻辑

### 弹幕导入/导出

- 导入功能转移至弹幕库右上角的菜单中
- 移除原有的导入页面
- 允许导入目录和zip文件
    - 导入目录时保留导入时的目录结构（弹幕库中树状显示）
- 可以使用弹幕库的多选功能导出所有内容

### 播放

- 在视频下方显示弹幕密度条（可以在设置中关闭）

### 其他

- 重做”关于“页面
- 替换部分图片素材
- 优化首次使用体验

**Full Changelog**: https://github.com/Mr-Quin/danmaku-anywhere/compare/v1.3.0...v1.4.0

## v1.3.0 - 2025-09-01

## 特性
* 兼容MacCMS的搜索（实验功能，获取的弹幕暂时归类为本地弹幕）
  - 搜索失效时可以在设置中选择其他的API地址
* 根据弹幕内容，显示跳过OP，ED的按钮（可以在设置中关闭此功能）
* 视频定位后填充弹幕
  - 之前：清空屏幕，弹幕从右边重新开始
  - 现在：尝试让弹幕填满屏幕
* 自动模式下，视频播放到一半时预下载下一集的弹幕（仅限弹弹Play源）
* 悬浮面板中更直观的显示正在播放的弹幕
* 弹幕列表中增加搜索按钮
* 增加可以从悬浮面板（`+`按钮打开的窗口）中打开扩展配置页面的按钮
* 修改剧集标题映射时会显示提示
* 拖拽`+`按钮后记录位置

## 修复
* 修复弹幕显示在悬浮面板上层的问题
* 修复弹弹Play集数匹配

## 其他
* 优化加载提示UI
* 替换部分图片素材

**Full Changelog**: https://github.com/Mr-Quin/danmaku-anywhere/compare/v1.2.2...v1.3.0

## v1.5.0 - 2026-02-07

## 特性

- 支持配置自定义 LLM（OpenAI兼容 API），可以使用本地ai识别番剧标题
- 新增手动编辑和查看标题映射关系的功能
- 支持通过编写自定义 CSS 来自由控制弹幕外观
- 新增弹幕发射间隔以及弹幕间距的设置。适当允许重叠比例可以增加弹幕的显示密度
- 新增一键清空弹幕数据库的功能
- 新增设置备份与恢复功能

## 修复

- 修复腾讯视频搜索时无结果的问题
- 修复了弹幕尺寸计算错误以及长弹幕因换行导致的定位偏差问题
- 修复特定情况下弹幕数据库会损坏的问题 （数据库出现问题现在可以使用清空按钮重置）
- 修复“标题映射”对话框无法继续的问题
- 关闭的跳过按钮现在不再会重复出现
- 修复了导出的 XML 格式无法被弹弹Play 识别导入的问题
